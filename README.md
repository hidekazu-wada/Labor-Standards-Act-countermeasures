# 労務コンプライアンス改善プロジェクト

このフォルダは、会社の労務まわり（シフト・休憩・残業・タイムカードなど）を整えていくための資料置き場です。
「どのファイルを見ればいいか」で迷わないよう、目的ごとにフォルダを分けています。

---

## 🟢 役員会議で使う資料はこれ

**`doc/01_役員会議/労務まわりの現状確認と整えておきたいこと.md`**

役員会議ではこの1ファイルだけ開けば大丈夫です。専門用語を避けて、
「何を確認したか」「整えておきたい点」「進め方のお願い」までまとめてあります。
細かい根拠を聞かれたら、下の「02_社内資料」を見れば答えられます。

---

## いま何をしているか（ざっくり）

現場スタッフからの気づきをきっかけに、労務の書類と現場の運用がそろっているかを確認しました。
整えておきたい点が見つかっており、専門家（社労士）の確認を待ちつつ、次の役員会議で共有して
進め方を相談する段階です。詳しくは `PROJECT.md` を参照。

> **メモ:** このリポジトリには社内の情報が含まれます。GitHub に上げる場合は **必ず private** にしてください。
